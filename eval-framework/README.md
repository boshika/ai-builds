# Build 04 — Eval Framework

A lightweight evaluation framework for measuring LLM output quality — correctness, groundedness, and consistency.

## Concepts Covered

- LLM-as-judge evaluation
- Retrieval quality metrics (precision, recall, MRR)
- Groundedness and hallucination detection
- Regression testing for prompt changes
